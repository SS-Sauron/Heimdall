Heimdall

Always watching. Never sleeping. Ready to wake the dead.

Welcome to the Heimdall Web Flasher. You can install Heimdall directly to your ESP32 from your browser. Important: Check the "Erase device" box during installation if you are flashing for the first time or want to reset your Captive Portal credentials.

Standard Profile

Plug-and-play defaults for typical home networks.

  • Standard MQTT topics
  • No authentication on MQTT commands
  • Uses default device MAC address

Hardened Profile

Maximum OPSEC for hostile environments.

  • HMAC-derived stealth MQTT topics
  • Time-based One-Time Password (TOTP) auth
  • MAC Address Spoofing
  • Hostname Obfuscation

⚠️ Troubleshooting

Device not showing up in the list?

  • Cables: You are likely using a charge-only USB cable. You MUST use a data cable. Try another cable first!
  • Drivers: Your computer might be missing the USB-to-UART drivers. Download them here:
    • Silicon Labs CP2102 Drivers
    • WCH CH340 Drivers
  • Browser: This feature uses the Web Serial API. It only works in Chromium-based browsers like Google Chrome, Microsoft Edge, or Brave. (Firefox and Safari are not supported).
